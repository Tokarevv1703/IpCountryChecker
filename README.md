# IpCountryChecker

Консольное приложение на C#, которое:

- Загружает список IP-адресов из файла `ips.txt`
- Отправляет GET-запросы к API https://ipinfo.io/{ip}/json
- Определяет страны, представленные в файле, и количество IP по каждой стране
- Находит страну с максимальным числом IP
- Выводит города для этой страны

## Используемые технологии

- .NET Console App
- HttpClient
- Newtonsoft.Json

## Запуск

1. Убедитесь, что файл `ips.txt` находится рядом с `Program.cs`
2. Выполните команду dotnet run
